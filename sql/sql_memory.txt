------------------------------------------  MySQL 메모리 할당 구조
MySQL 서버는 글로벌 메모리 영역과 로컬 메모리 영역이 있다.
각 영역의 메모리를 적절하게 설정해야 MySQL Server 메모리 락 걸리는 현상을 방지할 수 있다.

글로벌 메모리 영역
스레드 수와 무관하게 하나의 메모리 공간만 할당된 영역이다. 글로벌 메모리 영역은 모든 스레드에 공유됨.

로컬 메모리 영역
세션 메모리 영역이라고도 표현하는데, 공유되지 않고, 스레드 1개가 점유하는 영역이기 때문이다.


SQL 튜닝이나 인덱스 설계를 잘 갖춰서 로컬 메모리 영역을 보다 적게 사용하는 것이 바람직하다.
