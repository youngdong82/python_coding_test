문제 1. 모든 레코드 조회하기
SELECT *  모든 것을 조회
FROM ANIMAL_INS  ANIMAL_INS에서
ORDER BY ANIMAL_ID  ANIMAL_ID 순서대로

문제 2. 역순 정렬하기
SELECT NAME, DATETIME  NAME과 DATETIME을 조회
FROM ANIMAL_INS  ANIMAL_INS에서
ORDER BY ANIMAL_ID DESC   ANIMAL_ID 역순으로

순서대로는 ASC, 역순은 DESC

문제 3. 아픈 동물 찾기
SELECT ANIMAL_ID, NAME  ANIMAL_ID와 NAME을 조회
FROM ANIMAL_INS  ANIMAL_INS에서
WHERE INTAKE_CONDITION = 'Sick'  INTAKE_CONDITION이 ‘Sick’인 것을
ORDER BY ANIMAL_ID  ANIMAL_ID 순서대로

문제 4. 어린 동물 찾기
SELECT ANIMAL_ID, NAME  ANIMAL_ID와 NAME을 조회
FROM ANIMAL_INS  ANIMAL_INS에서
WHERE INTAKE_CONDITION <> 'Aged' INTAKE_CONDITION이 ‘Aged’가 아닌 것을
ORDER BY ANIMAL_ID  ANIMAL_ID 순서대로

아닌 것을 나타낼 때 <>나 != 중 하나를 쓰면 된다.
퍼포먼스를 위해선 <>를 쓰자.
