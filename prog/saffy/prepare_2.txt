응용 수리
  거리-속력-시간
  거리 = 속력 * 시간
  하나로 통일시키자!
  그림을 그려서 문제 이해를 직관적으로 하자.

    1. 속력 변경
      전체 거리가 L이고, 초기 속력이 a일 때,
      길이 x만큼 이동 후 속력 b로 y만큼 더 이동할 때의 전체 이동 시간을 구하는 문제가 기본적인 문제

      x/a + (L-x)/b  = 시간

    2. 평균 속력 (조화평균)
      전체 거리가 L인 도로를 왕복할 때,
      속력이 갈때는 a, 돌아올 때는 b이면, 평균 속력을 구하는 문제가 기본적인 문제

      왕복 거리 = 갈 때 거리 + 올 때 거리
      평균 속력 = 왕복 거리 / 총 시간
              = 2L / (갈때 시간 + 올 때 시간)
              = 2L / (L/a + L/b)
              = 2ab / (a+b)
            
    3. 보트
      강의 속력 a와 배의 속력 b가 주어졌을 때, 상황에 따라 속력이 달라지는 문제

      강과 진행방향이 같다면 a+b
      강과 진행방향이 틀리다면 a-b

    4. 따라잡기
      A와 B가 달리기를 할 때, 동시에 출발하면 t 시간 후에 A가 B를 따라잡습니다.
      B가 L만큼 앞에서 출발하고, A의 속력은 a, B의 속력은 b일 때 따라잡는 시간을 구하는 문제

      A의 거리 = B의 거리
      at = bt + L

    5. 만나기
      걷는 속력이 a인 A와 속력이 b인 B가 L만큼 떨어져 있습니다.
      서로 마주보며 걸어올 때, 만나는 시간을 구하는 문제

      총 거리 = A의 거리 + B의 거리
      L = at + bt
      L = t(a+b)
      t = L/(a+b)

      다른 방법으로,
      총 시간 = 총 거리 / 총 속력
      T = L / (a+b)

    6. 운동장
      한 바퀴 거리가 L인 운동장에서
      A 속력은 a, B 속력은 b이고 동시에 출발할 때
      6-1. 같은 방향으로 달려서 A가 1바퀴를 더 달려서 따라잡는 경우를 구하는 문제 유형
        a가 간 거리  = b가 간 거리 + L
        at = bt + L
      6-2. 반대 방향으로 달려서 A와 B가 만나는 경우의 문제 유형
        L = at + bt

    7. 기차
      기차의 길이가 a, 속력은 v, 터널은 b일 때
      7-1. 터널을 지나는 시간
      7-2. 마주오는 기차의 길이가 c이고 속력이 m일 때,
           두 기차가 완전히 지나치는 시간

  경우의 수
    중복의 경우를 고려하여 계산
    두 가지 이상의 사건이 이어서 일어나는 경우, 각 사건의 경우의 수 곱하기.

  순열
    n개의 원소에서 서로 다른 r개를 '순서를 고려'해서 배열하는 것. (0<= r <=n)
    nPr  = n! / (n-r)!

    예시) 5명 중 반장, 부반장 뽑는 경우의 수
      5*4*3*2*1 / 3*2*1
      120/6
      = 20

  조합
    서로 다른 n개의 원소에서 '순서를 고려하지 않고' r개를 택하는 조합을 찾는 것. (0<=r<=n)
    nCr =  nPr/r! =  n!/r!(n-r)!
    
    예시) 5명 중 대표 2명을 뽑는 경우의 수
      5*4*3*2*1 / 3*2*1
      120 /  (2*1)* (3*2*1)
      120 / 12
      = 10

  팩토리얼
    !은 팩토리얼
    n!은 1부터 n까지의 수를 모두 곱한 값.

    원형 탁자에 n명을 앉히는 경우의 수

  확률
    수학적으로 확률은
    특정 사건의 경우의 수를 전체 사건의 경우의 수로 나눈 것을 의미한다.

    특징
      언제나 0이상 1이하
      반드시 일어나는 일이면 1
    
    계산
      덧셈정리: a의 경우의 수 + b의 경우의 수 - a와 b의 교집합
      여집합: 1 - a의 경우의 수
      조건부 확률:
        사건 A, 사건 B에 대하여 A가 일어났을 때, B가 일어난 확률
        p(B|A) = A와 B의 교집합 / A의 경우의 수
  
      A와 B가 동시에 일어날 확률:
        A와 B의 교집합

  날짜,요일
    윤년은 366일, 나머지는 365일

    언제가 윤년인가?
      4로 나누어 떨어지는 해 중에서
      100으로 나누어 떨어지고
      400으로 나누어 떨어지면 윤년
    
    오늘을 포함하는지 확인

    모듈러 연산을 사용
      요일은 총 7가지의 경우가 있다는 것을 이용하여, 7로 나눈 나머지를 사용
      특정 날짜의 요일이 a요일 일 때, n일 후의 요일은, n을 7로 나눈 나머지가 b일 때, n일 후의 요일은 a+b요일
      특정 날짜의 요일이 a요일 일 때, n일 전의 요일은, -n을 7로 나눈 나머지가 -b일 때, n일 후의 요일은 a-b요일
      연단위의 계산

      2020년 4월 20일이 월요일이면,
      ex 1) 100일 후 요일은?
        100 % 7 = 2
      ex 2) 100일 전 요일은?
        -100 % 7 = -2
